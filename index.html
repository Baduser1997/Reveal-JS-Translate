Reveal.js

El Framework Para crear Presentaciones

Traducido por ISC JUAN LUIS / @UnTalMapayork

Hola

reveal.js le permite crear hermosas plataformas de diapositivas interactivas usando HTML. Esta presentación le mostrará ejemplos de lo que puede hacer.

Diapositivas verticales

Las diapositivas pueden anidarse una dentro de la otra.

Utilizar el Espacio clave para navegar a través de todas las diapositivas.


Sótano Nivel 1

Las diapositivas anidadas son útiles para agregar detalles adicionales debajo de una diapositiva horizontal de alto nivel.

Sotano Nivel 2

Eso es todo, es hora de volver a subir.


Slides

No es un Programador? No es un problema. Hay un editor visual con todas las funciones para crear estos, pruébelo en http://slides.com.

Punto de vista

Presione ESC para ingresar a la vista general de la diapositiva.

Mantenga presionada la tecla Alt y haga clic en cualquier elemento para acercarlo utilizandozoom.js. Alt + clic en cualquier lugar para alejar.

Touch optimizado

Las presentaciones se ven muy bien en dispositivos táctiles, como teléfonos móviles y tabletas. Simplemente desliza tus diapositivas.

Fondos de diapositivas

Set data-background="#dddddd" en una diapositiva para cambiar el color de fondo. Todos los formatos de color CSS son compatibles.

Imagenes de fondo

<section data-background="image.png">

Fondos de azulejos

<section data-background="image.png" data-background-repeat="repeat" data-background-size="100px">

Videos de Fondo

<section data-background-video="video.mp4,video.webm">

... y GIFs!

Transiciones de fondo

Diferentes transiciones de fondo están disponibles a través de la opción backgroundTransition. Este se llama "zoom". 

Reveal.configure({ backgroundTransition: 'zoom' })

Trancisiones de fondos

Puede anular las transiciones de fondo por diapositiva. 

<section data-background-transition="zoom">

  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) {
        node.className += ' roll';
      }
    }
  }
}

Estado global.

Set data-state="something" en una diapositiva y "something" se agregará como una clase al elemento del documento cuando la diapositiva esté abierta. Esto te permite aplicar cambios de estilo más amplios, como cambiar el fondo de la página.

Eventos de estado

Además, los eventos personalizados se pueden desencadenar por diapositiva vinculando al data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Tomate Un Momento

Presione B o . en tu teclado para pausar la presentación. Esto es útil cuando estás en el escenario y quieres sacar diapositivas que te distraigan de la pantalla..
